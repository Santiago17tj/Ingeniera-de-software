PASOS RÁPIDOS:
═══════════════════════════════════════════════════════════════

PASO 1: CREAR BASE DE DATOS
───────────────────────────
1. Abre tu panel de base de datos (phpMyAdmin, etc.)
2. Crea una nueva base de datos: reserva_espacios
3. Selecciona la base de datos
4. Abre el archivo: base_de_datos.sql
5. Copia TODO el contenido y pégalo en el SQL del panel
6. Ejecuta el SQL

PASO 2: CONFIGURAR
──────────────────
1. Extrae el ZIP en el directorio del subdominio
2. Abre el archivo: config.php
3. Cambia estos valores:
   - DB_USER: tu usuario de MySQL
   - DB_PASS: tu contraseña de MySQL
   - DB_NAME: reserva_espacios (o el nombre que le diste)

PASO 3: VERIFICAR
──────────────────
1. Visita tu subdominio
2. Deberías ver la página de inicio
3. Haz clic en "Iniciar Sesión"
4. Usa estas credenciales:
   Usuario: admin
   Contraseña: admin123

⚠️ IMPORTANTE: Cambia la contraseña del admin después del primer acceso
