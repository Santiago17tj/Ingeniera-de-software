═══════════════════════════════════════════════════════════════
  PASO 1: Actualizar la Base de Datos
═══════════════════════════════════════════════════════════════

Si ya tienes una base de datos con reservas existentes, necesitas
agregar los nuevos campos a la tabla.

OPCIÓN A: Usar el script PHP (Recomendado)
───────────────────────────────────────────
1. Abre tu navegador y ve a:
   http://localhost/reserva_espacios/agregar_campos_reserva.php

2. El script verificará si los campos existen y los agregará
   automáticamente si es necesario.

3. Una vez completado, ELIMINA el archivo agregar_campos_reserva.php
   por seguridad.

OPCIÓN B: Usar phpMyAdmin
──────────────────────────
1. Abre phpMyAdmin (http://localhost/phpmyadmin)

2. Selecciona tu base de datos (reserva_espacios)

3. Ve a la pestaña "SQL"

4. Copia y pega el contenido de agregar_campos_reserva.sql

5. Ejecuta el script

═══════════════════════════════════════════════════════════════
  PASO 2: Probar el Formulario
═══════════════════════════════════════════════════════════════

1. Inicia sesión en la aplicación

2. Ve a "Nueva Reserva"

3. Verifica que aparezcan los nuevos campos:
   - Nombre Completo (obligatorio)
   - Número de Teléfono (obligatorio)
   - Número de Identificación (obligatorio)

4. Completa el formulario y crea una reserva

5. Verifica que la información se guarde correctamente
